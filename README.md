# vue-js-ex

간단한 Vue.js 기반 프론트엔드 연습용 프로젝트입니다.  
Vue 컴포넌트, SPA 구조, 상태 관리 없이도 동작 가능한 기본 UI/뷰 구조 연습, JavaScript + HTML + CSS 기반으로 Vue의 핵심 개념을 실습해보는 목적을 갖고 있습니다.

## 🎯 프로젝트 목적

- Vue.js 기본 사용법 및 컴포넌트 구조 학습  
- HTML + CSS + JS + Vue 조합 개발 흐름 실습  
- SPA, 데이터 바인딩, 반응형 UI 개념 연습  
- Vue 초심자에게 적합한 간단한 예제 프로젝트  

## 📁 기본 프로젝트 구조 예시

```text
ue-js-ex/
├─ src/ ← Vue 소스 (components / assets / main.js 등)
├─ public/ ← 정적 파일 (index.html, favicon 등)
├─ package.json ← 프로젝트 의존성 및 스크립트 설정
└─ README.md ← 프로젝트 설명 파일
```

(실제 구조는 위와 유사하며, Vue CLI 또는 Vite 설정이 포함될 수 있습니다.)

## 🛠 사용 기술 & 환경

- Vue.js (JS 기반) — SPA 혹은 간단한 컴포넌트형 UI  
- HTML, CSS, JavaScript  
- Optional: Vue Router, 상태 관리(lib) 없이 순수 Vue 학습
